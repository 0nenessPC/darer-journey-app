# D.A.R.E.R. Journey

A React + Vite version of the D.A.R.E.R. Journey app.

## Run locally

1. Open a terminal in this folder:
   `C:\Users\Administrator\.openclaw\workspace\darer-journey`
2. Install dependencies:
   `npm install`
3. Start the dev server:
   `npm run dev`
4. Open the local URL shown in the terminal.

## Notes

- The app is currently runnable without external API keys.
- The intake flow and progression are implemented locally so you can test the experience immediately.
- If you want, I can wire the Dara chat back to Anthropic or OpenAI afterward.
